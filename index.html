Delimitadores

Delimitador padrão: aspas simples ou duplas (costuma-se usar aspas simples)

Não se coloca aspas em números

Entre parenteses coloca-se a função que se quer utilizar

Exemplo: print('Olá, mundo!'), retorna Olá, mundo!

print(7+4), retorna 11.

se usar print('7'+'4'), retorna 74, como string
